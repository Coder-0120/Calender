# 📅 Interactive Wall Calendar (React)

A responsive and interactive wall calendar component built using React.  
Inspired by a physical wall calendar design, this project combines clean UI with useful features like date range selection and note-taking.

## ✨ Features

- 📆 **Dynamic Calendar**
  - Automatically generates days for each month
  - Correct weekday alignment

- 🔵 **Date Range Selection**
  - Select start and end dates
  - Highlighted range with clear visual states

- 📝 **Notes Section**
  - Add notes for the month
  - Data persists using `localStorage`

- 🔄 **Month Navigation**
  - Easily switch between months and years

- 📱 **Responsive Design**
  - Works smoothly on desktop and mobile devices

---

## 🛠 Tech Stack

- React (Functional Components + Hooks)
- CSS (Custom styling, no frameworks)
- JavaScript (ES6+)
